# Soluciones

Este directorio contiene las soluciones de referencia para los ejercicios desarrollados en las distintas sesiones del curso **Fundamentos de SQL**.

## Propósito

Las soluciones se proporcionan como material de apoyo para:

- Verificar resultados.
- Comparar enfoques de resolución.
- Comprender buenas prácticas en SQL.
- Favorecer el aprendizaje autónomo.

## Contenido

| Archivo | Descripción |
|----------|-------------|
| Solucion-Ejercicio-Sesion_1 | Solución de los ejercicios de la Sesión 1 |
| Solucion-Ejercicio-Sesion_2 | Solución de los ejercicios de la Sesión 2 |
| Solucion-Ejercicio-Sesion_3 | Solución de los ejercicios de la Sesión 3 |
| Solucion-Ejercicio-Sesion_4 | Solución de los ejercicios de la Sesión 4 |

## Recomendación

Se recomienda intentar resolver cada ejercicio antes de consultar la solución. El aprendizaje de SQL se fortalece mediante la práctica y la experimentación directa con las consultas.

## Licencia

Este material forma parte de un Recurso Educativo Abierto (OER) destinado a apoyar la enseñanza y el aprendizaje de bases de datos y SQL.
