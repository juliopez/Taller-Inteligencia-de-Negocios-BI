# Taller Práctico de Fundamentos de Ingeniería de Datos y SQL

**Duración:** 18 horas  
**Modalidad:** Online sincrónica  
**Docente:** Dr. Julio Lopez-Nunez

## Descripción
El presente taller tiene como propósito fortalecer las competencias prácticas asociadas al modelamiento relacional, consultas SQL y fundamentos de ingeniería de datos, mediante actividades aplicadas desarrolladas en entornos cloud y ejercicios orientados a escenarios organizacionales reales.

## Aprendizajes Esperados

| Resultados de Aprendizaje | Descripción                                                                                                                                                                                                                    |
| ------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| RAA1.                     | Diseñar modelos de datos relacionales utilizando principios de la ingeniería de datos y lenguaje SQL, con el fin de estructurar información de manera eficiente en contextos organizacionales y proyectos intensivos en datos. |
| RAA2.                     | Implementar consultas SQL para la extracción, transformación y análisis de datos desde sistemas de almacenamiento estructurado, considerando criterios de eficiencia y coherencia con los requerimientos del proyecto.         |
| RAA3.                     | Evaluar estructuras de almacenamiento de datos aplicando conceptos de data warehousing en escenarios reales de ingeniería de datos, valorando su impacto en la calidad, disponibilidad y uso estratégico de la información.    |

## Metodología

El taller tendrá un enfoque predominantemente práctico, basado en workshops guiados, resolución de ejercicios y construcción progresiva de soluciones utilizando SQL y herramientas cloud orientadas a ingeniería de datos.

Las actividades prácticas serán desarrolladas utilizando escenarios académicos y organizacionales orientados al análisis y gestión de datos.
