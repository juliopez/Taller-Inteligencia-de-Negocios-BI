# Fundamentos SQL

Material introductorio orientado al modelamiento relacional, consultas SQL y fundamentos de almacenamiento estructurado de datos, utilizando ejercicios prácticos y entornos cloud.

## Contenidos

- Modelamiento relacional
- Entidades y atributos
- PK/FK
- Cardinalidad
- CRUD
- SELECT
- JOIN
- GROUP BY
- Introducción a DW y BI

## Tecnologías utilizadas

- Oracle APEX
- SQL Workshop
- AWS Academy (conceptual)
- Herramientas de diagramación online

## Enfoque del taller

El taller posee un enfoque eminentemente práctico, centrado en la resolución de ejercicios de modelamiento relacional y consultas SQL utilizando casos contextualizados.

## Estructura

- Sesión 1: Modelamiento relacional
- Sesión 2: SQL básico y CRUD
- Sesión 3: JOIN y agregaciones
- Sesión 4: Consultas aplicadas
- Sesión 5: Introducción a DW y ecosistemas BI
