# Orientaciones Evaluación 1

---

### 1. Forma de entrega

La entrega se realizará a través de la plataforma del curso.

Dado que el sistema no está configurado para entregas grupales, **cada integrante del grupo deberá subir el mismo archivo correspondiente al proyecto**.

El documento debe ser entregado en **formato PDF**.

Se recomienda que el archivo incluya claramente:

* Nombre del proyecto
* Integrantes del grupo

---

### 2. Alcance de la Evaluación 1

Esta primera entrega corresponde a una etapa **conceptual y de comprensión de los datos**, por lo que no se espera aún el uso de herramientas de software ni la implementación de procesos técnicos.

En particular, el trabajo debe enfocarse en:

* Definición clara del problema analítico
* Formulación de objetivos del proyecto
* Planteamiento de preguntas que guiarán el análisis
* Selección y comprensión del conjunto de datos
* Revisión del diccionario de datos
* Análisis preliminar de las variables disponibles
* Identificación de posibles datos relevantes para el análisis
* Propuesta inicial del hecho y dimensiones (a nivel conceptual)

---

### Importante

En esta etapa:

* **NO se requiere implementación de ETL**
* **NO se requiere construcción del modelo dimensional completo**
* **NO se requiere uso de herramientas de Business Intelligence**

El foco está en comprender el problema y los datos.

---

### 3. Sobre el video explicativo

Para esta primera evaluación, **no se solicitará el video explicativo**, dado que el trabajo aún no considera desarrollo técnico.

Este elemento será considerado en las siguientes evaluaciones del proyecto, donde sí se implementarán procesos y herramientas.

---

### 4. Recomendaciones

Para desarrollar un buen trabajo, se sugiere:

* Revisar cuidadosamente el diccionario de datos
* Asegurar coherencia entre el problema planteado y los datos disponibles
* Evitar definir soluciones antes de comprender el dataset
* Justificar las decisiones tomadas en base a los datos

---

### 5. Consideración final

Recuerden que este proyecto es progresivo, y que esta primera etapa es fundamental, ya que define la base sobre la cual se desarrollarán las siguientes evaluaciones.
