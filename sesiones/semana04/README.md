# Semana 4 — Taller Práctico: ETL Básico con SSIS

En esta sesión, los estudiantes desarrollarán su primer pipeline de datos en un contexto de Business Intelligence, implementando un proceso básico de **Extracción y Carga (E-L)** utilizando SQL Server Integration Services (SSIS).

---

## Objetivo de la sesión

Comprender cómo fluye la información desde distintas fuentes hacia un sistema centralizado, mediante la implementación de un proceso de integración de datos.

---

## Contexto

En las sesiones anteriores se abordaron los fundamentos de Business Intelligence y los problemas asociados a la gestión de datos en las organizaciones, tales como:

- Dispersión de datos  
- Inconsistencias  
- Problemas de calidad  
- Falta de historial  
- Limitaciones de rendimiento  

En esta sesión, se implementa una solución inicial a estos problemas mediante un proceso de tipo **E-L (Extract - Load)**.

## Actividad práctica

Durante la sesión, los estudiantes deberán:

1. Conectarse a una instancia de AWS con entorno Windows  
2. Utilizar Visual Studio Data Tools  
3. Implementar un flujo de datos en SSIS  
4. Extraer información desde una fuente  
5. Cargar los datos en un destino  

---

## Alcance

Este ejercicio se enfoca únicamente en:

✔ Extracción de datos  
✔ Carga de datos  

❌ No incluye transformaciones complejas (ETL completo)

---

## Relación con el curso

Este taller representa el primer acercamiento práctico a la construcción de pipelines de datos, y servirá como base para:

- Modelado dimensional
